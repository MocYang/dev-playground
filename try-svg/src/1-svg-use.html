在Web浏览器中使用SVG

1: iframe

        <iframe
          src="../assets/images/iconmonstr-computer-12.svg"
          frameborder="0"
          width="300"
          height="300"
        ></iframe>

2: img

作为 img src 属性引入
        <img
          src="../assets/images/iconmonstr-computer-12.svg"
          width="300"
          height="300"
        ></img>

3: background

作为背景图片
        
          .svg__bg {
            width: 300px;
            height:300px;
            background: url("../assets/images/iconmonstr-computer-12.svg") no-repeat;
          }

NOTE:

动态的SVG（结JS生成）作为背景时的坑：不能直接创建SVG元素字符串直接赋值给元素的背景属性。需要window.btoa('svg')转义

          let svg = `
                      <svg
                        version="1.1"
                        id="Layer_1"
                        xmlns="http://www.w3.org/2000/svg"
                        xmlns:xlink="http://www.w3.org/1999/xlink"
                        x="0px"
                        y="0px"
                        width="${contentWidth / 2}px"
                        height="${contentHeight}px"
                        viewBox="0 0 ${contentWidth / 2 + ' ' + contentHeight / 2}"
                        enable-background="new 0 0 ${contentWidth * 2 + ' ' + contentHeight}"
                        xml:space="preserve"
                      >
                        <text
                           transform="rotate(-32)"
                           x="${0}"
                           y="${contentHeight / 3}"
                           font-size="${mockFontSize}"
                           fill="#aaa"
                          >${name + '/Confidential'}
                        >/text<
                      </svg>`;
          let encoded = window.btoa(svg);
          waterWall.style.background = "url(data:image/svg+xml;base64,"+encoded+")";
